# ⚡ Калькулятор Электрика BY DIMA XP

Мощный, простой и удобный инструмент для инженеров-электриков, проектировщиков, электромонтажников и радиолюбителей. Настольное приложение автоматизирует рутинные расчеты параметров электрических сетей в соответствии с электротехническими стандартами.

## 🚀 Возможности

### 1. Расчет автоматических выключателей
Проверка поведения теплового и электромагнитного расцепителей автоматических выключателей при перегрузках согласно стандартным время-токовым характеристикам:
* **Условный ток нерасцепления (1.13 * In):** проверка работы сети под длительной нагрузкой без ложных срабатываний.
* **Условный ток расцепления (1.45 * In):** расчет времени гарантированного отключения автомата при перегрузке.
* **Ток надежного срабатывания (2.55 * In):** экспресс-анализ аварийного отключения при сильных перегрузках.

### 2. Базовые электротехнические расчеты
* **Закон Ома:** взаимосвязь напряжения (U), силы тока (I), сопротивления (R) и мощности (P).
* **Конвертер мощности и тока (Ток / Ватты):** быстрый пересчет Ватт в Амперы и обратно для однофазных (220В) и трехфазных (380В) сетей.
* **Дополнительные утилиты:** быстрый расчет базовых параметров цепей для подбора оптимального оборудования.

## ⚙️ Быстрый старт и сборка

### Требования
* Установленная среда разработки Visual Studio 2013, JetBrains Rider или .NET SDK.

### Сборка и запуск
1. Клонируйте репозиторий на свой компьютер:
   ```bash
   git clone https://github.com
   ```
2. Откройте проект в Visual Studio (через файл решения или файл проекта `Calc Watt To Amper BY DIMA XP.csproj`).
3. Нажмите кнопку **Пуск (Start / F5)** для компиляции и запуска приложения.

## 📖 Использование
1. Запустите приложение `Electro-Calc`.
2. В секции **Закон Ома** или **Ток и Ватты** введите известные параметры, чтобы программа автоматически рассчитала остальные величины.
3. В секции **Расчет автоматов** введите номинальный ток автомата (In). Программа мгновенно отобразит пороговые значения токов перегрузки (1.13 * In, 1.45 * In, 2.55 * In, 5.00 * In, 10.00 * In) для оценки защиты кабельных линий.
